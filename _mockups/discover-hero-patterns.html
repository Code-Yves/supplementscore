Discover hero — subtle background patterns

Four background-pattern directions to break up the dark-teal block. All keep the existing coral stat + content layout intact; only the layer behind the content changes. Same idea as the Articles page patterns, but tuned for Discover’s “wild facts” tone.

A

Trial growth line + scatter dots (Articles-page family)

Faint ascending data-viz line with scattered points — visually says “trial evidence” without being literal. Layered with a very low-opacity grid (same trick the Articles hero uses). Most consistent with the Articles hero family.
