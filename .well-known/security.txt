# Reporting
# Please report security vulnerabilities responsibly. Contact via the
# email listed on /about.html. Allow up to 7 days for an acknowledgement.

# Scope
# - The site itself (HTML, CSS, JS, sitemap, robots.txt)
# - The open-data files at /data/
# - Anything that could lead to medical-data exposure or content injection

# Out of scope
# - GitHub Pages infrastructure (report to GitHub)
# - Formspree (report to Formspree)
# - Google Fonts CDN (report to Google)
